Implementa un header tipo respond.io con mega menu y comportamiento idéntico:

Requisitos funcionales (no negociables)

Header sticky arriba con fondo oscuro y leve transparencia + blur (backdrop-filter), z-index alto.

Links principales: Product (dropdown), Industries, Resources (dropdown), Pricing, Why Us.

Lado derecho: icono globo, Login, botón secundario Talk to Sales, botón primario azul Start Free Trial.

Los dropdowns son mega menus:

Abren al hover en desktop (con delay corto 80–120ms) y también al click (por accesibilidad).

Cierran al salir del área, ESC, click afuera.

Navegables por teclado (tab/shift-tab/flechas) y con ARIA correcto.

El panel del mega menu:

Ancho grande (tipo 960–1100px), centrado bajo el header.

Fondo oscuro, border sutil, sombra profunda, esquinas redondeadas grandes (16–20px).

Grid interno: 4 columnas para “Product” y para “Resources”.

En “Product”, última columna incluye “Success Stories / WhatsApp Business Calling API / Channels and Integrations” y debajo una fila de iconos circulares + “40+”.

En “Resources”, última columna es una tarjeta “Partners” con CTA.

Responsive:

Desktop: mega menu.

Mobile: menú hamburguesa con acordeones (sin mega menu flotante).

Recomendación técnica (para hacerlo bien y rápido)

Usar Radix UI NavigationMenu (vía shadcn/ui) para accesibilidad y manejo de focus.

React + Tailwind. Nada de CSS caótico inline.

Menú data-driven: un objeto JSON define columnas/items para Product y Resources.

Estilos clave:

Header: backdrop-blur, bg-black/60, borde inferior sutil.

Panel: bg-[#0b0f19]/95, border-white/10, shadow fuerte, rounded-2xl.

Items: icono en contenedor cuadrado bg-white/5, hover bg-white/10, texto secundario gris.

Definition of Done (cómo verifico)

Se ve igual que respond.io en desktop en 1440px.

Hover abre/cierra como en respond.io.

Tab + Enter + ESC funciona perfecto.

No hay “jank”: transición suave (150–220ms), sin saltos.

Esqueleto de implementación (Next.js/React + Tailwind + shadcn)
1) Instalar shadcn NavigationMenu
npx shadcn@latest add navigation-menu
npx shadcn@latest add button

"use client";

import * as React from "react";
import Link from "next/link";
import { Globe } from "lucide-react";
import {
  NavigationMenu,
  NavigationMenuContent,
  NavigationMenuItem,
  NavigationMenuLink,
  NavigationMenuList,
  NavigationMenuTrigger,
  navigationMenuTriggerStyle,
} from "@/components/ui/navigation-menu";
import { Button } from "@/components/ui/button";
import { PRODUCT_COLUMNS, RESOURCES_COLUMNS, MegaColumn } from "./nav-data";

function MegaPanel({
  columns,
  rightCard,
}: {
  columns: MegaColumn[];
  rightCard?: React.ReactNode;
}) {
  return (
    <div className="w-[1040px] p-6">
      <div className="grid grid-cols-4 gap-10">
        {columns.slice(0, 3).map((col) => (
          <div key={col.heading} className="space-y-4">
            <div className="text-xs font-semibold tracking-wide text-white/60">
              {col.heading}
            </div>
            <div className="space-y-2">
              {col.items.map((it) => (
                <Link
                  key={it.title}
                  href={it.href}
                  className="group flex items-center gap-3 rounded-xl px-2 py-2 transition hover:bg-white/5 focus:outline-none focus:ring-2 focus:ring-white/20"
                >
                  <div className="flex h-9 w-9 items-center justify-center rounded-lg bg-white/5 ring-1 ring-white/10 transition group-hover:bg-white/10">
                    {it.icon ? <it.icon className="h-4 w-4 text-white/70" /> : null}
                  </div>
                  <div className="leading-tight">
                    <div className="text-sm font-medium text-white/90">
                      {it.title}
                    </div>
                    {it.description ? (
                      <div className="mt-0.5 text-xs text-white/50">
                        {it.description}
                      </div>
                    ) : null}
                  </div>
                </Link>
              ))}
            </div>
          </div>
        ))}

        <div className="space-y-4">
          <div className="text-xs font-semibold tracking-wide text-white/60">
            {columns[3]?.heading}
          </div>

          <div className="space-y-2">
            {columns[3]?.items?.map((it) => (
              <Link
                key={it.title}
                href={it.href}
                className="group flex items-center gap-3 rounded-xl px-2 py-2 transition hover:bg-white/5 focus:outline-none focus:ring-2 focus:ring-white/20"
              >
                <div className="flex h-9 w-9 items-center justify-center rounded-lg bg-white/5 ring-1 ring-white/10 transition group-hover:bg-white/10">
                  {it.icon ? <it.icon className="h-4 w-4 text-white/70" /> : null}
                </div>
                <div className="text-sm font-medium text-white/90">{it.title}</div>
              </Link>
            ))}
          </div>

          {rightCard ? (
            <div className="mt-4 rounded-2xl border border-white/10 bg-white/5 p-4">
              {rightCard}
            </div>
          ) : null}
        </div>
      </div>
    </div>
  );
}
